# Media-App-Rattrapage
Projet DisneyFlix


Description :
DisneyFlix est un site web permettant de parcourir une sélection de films Disney. Le projet utilise HTML, SCSS et Vite.


Prérequis :
Node.js installé
pnpm installé

Installation :
Cloner le dépôt sur votre machine
Accéder au dossier du projet cloné
Installer les dépendances avec : pnpm install

Lancement du serveur :
Démarrer le serveur de développement avec : pnpm run dev
